# Modify 🎵

Modify is an AI-powered music recommendation web app that detects your facial expression and plays music based on your current mood.

## Features

- 😊 Face expression detection using MediaPipe
- 🎵 Mood-based music recommendations
- ☁️ Song & poster upload with ImageKit
- 🔐 User authentication
- 🎧 Built-in music player
- ⚡ Full-stack MERN application

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- MediaPipe
- ImageKit
- Axios

## Installation

```bash
git clone https://github.com/Suryansh10-ctrl/Modify.git
cd Modify
npm install
```

## Run

### Backend

```bash
npm run dev
```

### Frontend

```bash
npm run dev
```
